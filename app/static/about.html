Instructions

The app requires two slightly inconvenient things to access your games and make recommendations:

  • Your steam id (oof -- kind of a pain)
  • A steam profile set to public with shared game details

The second one is unavoidable. A few years ago, Valve changed the default profile settings to hidden. Here's how to change them back:

  1. Sign into Steam
  2. Click on your profile name in the upper right corner and select "View My Profile"
  3. Select "Edit Profile > "My Privacy Settings"
  4. Change profile visibility and game details to "public"
    (The app needs to know game playtime, so make sure you don't hide that!)
